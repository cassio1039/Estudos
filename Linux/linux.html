Comandos linux

Comandos básicos:

rm -rf: Apagar uma pasta sem pedir confirmação(exclusão direta).

rm -ri: Apagar uma pasta pedindo confirmação(exclusão com permissão).

mkdir: Criar uma pasta.

ls: Mostra os arquivos que existem no diretório onde você se encontra.

cd "nome da pasta": Avança até a pasta selecionada.

cd ".." : Volta uma pasta anterior ao diretório de onde você se encontra.

touch: Cria um arquivo no direitório onde você se encontra
