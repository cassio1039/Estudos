Comandos linux

Comandos básicos:

  • rm -rf nome-da-pasta: Apagar uma pasta sem pedir confirmação(exclusão direta).

  • rm -ri nome-da-pasta: Apagar uma pasta pedindo confirmação(exclusão com permissão).

  • mkdir nome-da-pasta: Criar uma pasta.

  • ls: Mostra os arquivos que existem no diretório onde você se encontra.

  • cd nome-da-pasta: Avança até a pasta selecionada.

  • cd ".." : Volta uma pasta anterior ao diretório de onde você se encontra.

  • touch nome-do-arquivo: Cria um arquivo no direitório onde você se encontra
