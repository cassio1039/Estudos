Comandos docker

  • docker compose up: Rodar a imagem em um container.

  • docker compose up -d (detached): Rodar a imagem do container em um segundo plano.

  • docker compose down: Destrói a imagem do container.

  • docker compose up --force-recreate: Força a recriação do container sem desliga-lo.

  • docker compose -f [path/arquivo] up: Sobe a imagem a partir de um arquivo compose.yaml no caminho especificado.

  • docker ps: Lista todos os containers rodando.

  • docker -ps -a: Lista todos os containers existentes, inclusive os que não estão em uso.

  • docker logs (nome do container): Retorna os logs do container especificado pelo nome

  • psql --host=localhost --username=postgres --port=5432: Conectar ao cliente psql do PostgreSQL.
