Git

Comandos git

Todos estes comandos funcionam de forma offline no Git e podem ser usados sem a necessidade de uma conexão com a internet, pois nenhum deles transmite informações para fora do seu computador.

  • git init: Inicia um repositório Git em um diretório local.

  • git clone: Clona um repositório remoto (como no GitHub).

    Ex: git clone https://github.com/cassio1039/meu-projeto.git

  • git status: Mostra os arquivos do diretório, novos arquivos, modificações e etc.

  • git add "arquivo": Sobe alterações para a staging area(área de preparação).

    Ex: git add index.html ou git add. (para todos)

  • git log: Listar os commits do repositório.

  • git commit: Realiza novos commits, subindo modificações

  • git commit -m "mensagem": Realiza novos commits, subindo modificações, porém você pode colocar o motivo do commit

  • git commit --amend: Substitui o commit anterior por um novo commit porém aproveitando as alterações do commit anterior

  • git diff: Mostra a diferença entre as versões e alterações dos arquivos

  • git push: Envia os arquivos do diretório local para área remota.

  • git pull: Puxa os quivos do diretório remoto para área local.

  • git merge: Junta outra branch com a branch atual.

    Ex: git merge nova-feature

  • git branch: Mostra todas as branches existente e o sinal de "*" mostra em qual branch você está.

  • git checkout/switch: Muda para a branch informada. ponto

  • git branch -m novo-nome: Renomear branch se você já estiver nela.

  • git branch -m nome-antigo nome-novo: Renomear branch se você estiver fora da branch que deseja renomear.
